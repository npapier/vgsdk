vgSDK, a shortcut for Versatile Graphical Software Development Kit, consists of a set of cross-platform C++ libraries which provides a complete framework for doing graphical related tasks.

# Online documentation #
generated by Doxygen
  * [0.5 API](http://www.digital-trainers.com/XP-FARM/doc_vgsdk_0-5/html) (trunk)
  * [0.4 API](http://vgsdk-doc.googlecode.com/svn/tags/0-4/html/index.html)

# Samples #
  * [Hello world](https://code.google.com/p/vgsdk/source/browse/trunk/examples/qt/simpleCube/src/main.cpp).

Create a simple Qt application with a vgSDK canvas drawing a cube with a directional light.

  * [Underlay and overlay](https://code.google.com/p/vgsdk/source/browse/trunk/examples/qt/underlayOverlay/src/main.cpp).

Underlay, overlay and cairo text demo.

  * [Simple tessellation](https://code.google.com/p/vgsdk/source/browse/trunk/examples/qt/simpleTessellation/src/main.cpp).

Sample to show how to enable adaptive phong tessellation. Criterion
uses is the number of pixels per edge desired (15 pixels in this case).

  * [Examples using Python](https://code.google.com/p/vgsdk/source/browse/trunk/examples/python)
vgSDK uses [swig](http://www.swig.org/) to generate a [Python](https://www.python.org/) binding. The first use case of this binding is the loading of python file in the vgsdkViewer to create and configure a scene graph. Several examples are available [here](https://code.google.com/p/vgsdk/source/browse/trunk/examples/python).
